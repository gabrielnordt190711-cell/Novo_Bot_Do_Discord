# Novo Bot do Discord

Um bot simples e divertido para Discord feito em Python.

## ✨ Comandos

| Comando            | O que faz                                      |
|--------------------|------------------------------------------------|
| `$duck`            | Envia foto aleatória de um pato               |
| `$check`           | Identifica imagem usando IA                    |
| `$moedas`          | Mostra cotação de moedas (USD, EUR, BTC)       |
| `$quantascores`    | Conta quantas cores tem uma imagem             |
| `$iniciarcarro`    | Inicia o jogo de adivinhar o carro             |

### 🎯 Como jogar Adivinhar Carro
Use `$iniciarcarro`, envie o nome do carro e tente acertar!
